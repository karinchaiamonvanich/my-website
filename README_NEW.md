# Web Portfolio

พอร์ตโฟลิโอเว็บไซต์ส่วนบุคคล สร้างด้วย HTML, CSS, และ JavaScript

## 🎯 คำอธิบาย

โปรเจกต์นี้เป็นเว็บไซต์พอร์ตโฟลิโอที่แสดงความสามารถ ประสบการณ์ และโครงการต่างๆ ออกแบบมาให้เป็นมืออาชีพและเป็นมิตรกับผู้ใช้

## ⚙️ เทคโนโลยีที่ใช้

- **HTML5** - โครงสร้างหน้าเว็บ
- **CSS3** - จัดรูปแบบและการตอบสนองต่ออุปกรณ์ต่างๆ
- **JavaScript** - การทำงานแบบไดนามิกและปฏิสัมพันธ์

## 🚀 วิธีการใช้

### 1. **เปิดเว็บไซต์**

**วิธีที่ 1: โดยใช้เบราว์เซอร์ (ง่ายที่สุด)**
- ไปที่โฟลเดอร์ `my-website`
- คลิกขวาที่ไฟล์ `index.html`
- เลือก "Open with" และเลือกเบราว์เซอร์ที่ต้องการ
- หรือลากไฟล์ `index.html` ลงไปในเบราว์เซอร์

**วิธีที่ 2: โดยใช้ Local Server (ถ้าใช้ VS Code)**
- ติดตั้ง Extension "Live Server" จาก VS Code
- คลิกขวาที่ไฟล์ `index.html`
- เลือก "Open with Live Server"

**วิธีที่ 3: โดยใช้ Command Line**
```bash
# ใช้ Python 3
python -m http.server 8000

# หรือใช้ Node.js (ถ้าติดตั้ง)
npx http-server
```

แล้วเปิดเบราว์เซอร์ไปที่ `http://localhost:8000` (หรือ port ตามที่แสดง)

### 2. **นำทางเว็บไซต์**

- **หน้าหลัก (Home)** - แสดงข้อมูลส่วนบุคคลและภาพรวม
- **โครงการ (Projects)** - คลิกที่โครงการต่างๆ เพื่อดูรายละเอียด
- **ลิงค์สำเร็จ** - คลิกที่ลิงค์เพื่อไปยังหน้าโครงการหรือหน้าแรก

### 3. **ตัวแปรและการทำงาน**

**ไฟล์ `index.html`**
- ประกอบด้วยโครงสร้างหลักของเว็บไซต์
- มีส่วน header, main content, และ footer
- เชื่อมโยงกับ CSS และ JavaScript

**ไฟล์ `style.css`**
- ควบคุมลักษณะการปรากฏของเว็บไซต์
- รวมการออกแบบ responsive สำหรับอุปกรณ์ต่างๆ
- ตัวแปร CSS สำหรับสี, ฟอนต์, และระยะห่าง

**ไฟล์ `script.js`**
- เพิ่มการทำงานแบบไดนามิก
- จัดการการคลิกและปฏิสัมพันธ์ผู้ใช้
- ติดตั้งรูปภาพและสร้าง DOM elements

## ✏️ การแก้ไขและปรับแต่ง

### เปลี่ยนข้อมูลส่วนบุคคล
- เปิดไฟล์ `index.html` ด้วยตัวแก้ไขข้อความ
- ค้นหาและแก้ไขชื่อ, ตำแหน่ง, และรายละเอียดอื่นๆ

### เพิ่มโครงการใหม่
- สร้างไฟล์ HTML ใหม่ในโฟลเดอร์ `projects/`
- ใช้ `template.html` เป็นแม่แบบ
- อัปเดตลิงค์ใน `index.html`

### ปรับแต่งสี และสไตล์
- เปิดไฟล์ `style.css`
- แก้ไขค่าสี, ขนาด, และ font ตามต้องการ

### เพิ่มรูปภาพ
- เพิ่มรูปภาพใหม่ไปยังโฟลเดอร์ `img/`
- อัปเดตการอ้างอิงในไฟล์ HTML และ CSS

## 🎨 คุณสมบัติ

✅ ออกแบบ Responsive - ทำงานบนโทรศัพท์, แท็บเล็ต, และเดสก์ท็อป
✅ นำทางแบบเรียบง่าย - เมนูสะดวกและปฏิสัมพันธ์ที่ดี
✅ โครงการหลายรายการ - แสดงความสามารถและประสบการณ์
✅ ไม่มี Dependencies - ไม่จำเป็นต้องติดตั้งแพคเกจเพิ่มเติม

## 📱 ความเข้ากันได้

- ✅ Chrome, Firefox, Safari, Edge
- ✅ มือถือ, แท็บเล็ต, เดสก์ท็อป

## 🔧 การแก้ไขปัญหา

**ไฟล์ไม่โหลด**
- ตรวจสอบให้แน่ใจว่าไฟล์อยู่ในโฟลเดอร์ที่ถูกต้อง
- ใช้ Local Server แทนการเปิดไฟล์โดยตรง

**ส่วนประกอบไม่แสดง**
- เปิด Developer Tools (F12 หรือ Right-click > Inspect)
- ตรวจสอบ Console สำหรับข้อความข้อผิดพลาด

**ปัญหาการแสดงผล**
- ล้างแคช (Ctrl + Shift + Delete หรือ Cmd + Shift + Delete)
- ลองใช้เบราว์เซอร์อื่น
